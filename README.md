CRAFT: Hybrid Demand Forecasting Framework

1.Overview
CRAFT is a hybrid demand forecasting framework developed for retail inventory prediction. 
The solution combines the strengths of LightGBM and Temporal Fusion Transformer (TFT) to improve 
forecasting accuracy across products with varying demand patterns.

2.Problem Statement
Accurate demand forecasting is critical for inventory management, reducing stockouts, minimizing overstock situations, and improving supply chain efficiency
Traditional forecasting methods often struggle with seasonal patterns, demand volatility, and complex temporal dependencies
